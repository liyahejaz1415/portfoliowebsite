# Internship Assignment: Responsive Portfolio Website

A clean, modern, and completely responsive personal portfolio website built as part of the Day 1 internship milestone.

## 🚀 Features Implemented (Day 1)
* **Project Architecture Setup:** Organized workspace with custom decoupled styling assets.
* **Responsive Navigation Bar:** Implemented a sticky header layout with a dynamic mobile drawer menu toggle.
* **Semantic Structure:** Drafted structural anchors matching the baseline layout blueprint.

## 🛠️ Built With
* HTML5 (Semantic Structure)
* Custom CSS3 (CSS Variables, Flexbox, CSS Grid layout controls)
* Vanilla JavaScript (Mobile navigation states)

## 🔧 Local Installation & Setup
1. Clone this repository or download the source code zip.
2. Open the directory on your computer.
3. Double-click `index.html` to execute and preview the interface natively inside your web browser.
